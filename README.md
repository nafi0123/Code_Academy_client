# Codenest

**Codenest** is a modern web application built with **Next.js (App Router)** and **Firebase Authentication**. It provides a secure and interactive platform for users to explore and manage content.

## Features

- User authentication using **Firebase Auth** (Email/Password, Google Sign-In, etc.)
- Modern **Next.js App Router** architecture
- Responsive and interactive UI
- Secure access to protected pages
- Real-time updates using Firebase backend

---

## Tech Stack

- **Frontend:** Next.js, React
- **Authentication:** Firebase Auth
- **Deployment:** Vercel
- **Styling:** Tailwind CSS
